Angular pipes are used in template expressions, denoted by the | symbol, to apply transformations to values. Pipes can be used to format data for presentation, filter lists, sort arrays, and perform various other operations without modifying the underlying data.

Built in pipes

{{ product | uppercase }}

{{ price | currency : "EUR" : "symbol" }}

{{ release_date | date : "fullDate" }}

{{ meta_data | json }}

{{ dob | agePipe}}
